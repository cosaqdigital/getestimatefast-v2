# GetEstimateFast

Static lead-generation website for local construction estimate requests in the Riverview, Florida area.

## Pages

- `index.html` - main landing page and service selector
- `bathroom-remodeling.html` - guided bathroom remodeling estimate form
- `flooring-installation.html` - guided flooring installation estimate form
- `painting.html` - guided painting estimate form
- `drywall.html` - guided drywall estimate form
- `thank-you.html` - post-submit confirmation and Google Ads conversion page

## Notes

- The site is plain HTML, CSS, and JavaScript.
- Forms submit to FormSubmit and include WhatsApp, SMS, and email backup links.
- No build step or backend is required.
