# GFPROD-006 — Playbook de Performance Front-end

Versão: 1.0
Status: Approved
Sistema: GF-OS
Projeto: GetEstimateFast

## Objetivo
Definir o padrão oficial de performance front-end para garantir rapidez, excelente experiência do usuário e suporte ao crescimento da plataforma.

## Princípios
Velocidade, simplicidade, eficiência, escalabilidade e melhoria contínua.

## Core Web Vitals
LCP <= 2.5s, INP <= 200ms e CLS <= 0.1.

## Estratégias
Lazy loading, code splitting, otimização de imagens, fontes, CSS, JavaScript e cache.

## Auditorias
Revisões antes de publicações relevantes, mensalmente e após alterações estruturais.

## Checklist
Core Web Vitals, imagens, fontes, scripts, CSS, cache e aprovação GF-OS.

## KPIs
LCP, INP, CLS, tamanho das páginas, peso de JS, peso de imagens e taxa de abandono.

## Conclusão
Este playbook estabelece o padrão oficial de performance front-end do GetEstimateFast.
