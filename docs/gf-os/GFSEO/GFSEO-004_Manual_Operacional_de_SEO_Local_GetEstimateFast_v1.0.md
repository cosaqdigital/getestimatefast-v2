# GFSEO-004 — Manual Operacional de SEO Local

Versão: 1.0
Status: Approved
Sistema: GF-OS
Projeto: GetEstimateFast

## Objetivo
Definir o padrão oficial para planejamento, criação, otimização e manutenção da presença local do GetEstimateFast.

## Estratégia
Expandir geograficamente de forma gradual, consolidando cidades prioritárias antes de novas regiões.

## Estrutura das Páginas
URL amigável, title, meta description, H1, conteúdo original, CTA, FAQ, links internos, breadcrumb, schema e otimização mobile.

## Conteúdo Local
Cada cidade deve possuir informações próprias, evitando conteúdo duplicado.

## Arquitetura
Serviço → Cidade, mantendo consistência entre todos os clusters.

## Critérios
Criar páginas locais somente quando houver demanda, potencial comercial e capacidade de produzir conteúdo original.

## Checklist
Validar URL, SEO on-page, schema, links internos, performance, revisão e aprovação GF-OS.

## Indicadores
Impressões, cliques, CTR, conversão, leads por cidade e cobertura geográfica.

## Conclusão
O Manual Operacional de SEO Local estabelece o padrão para expansão regional consistente e sustentável do GetEstimateFast.
