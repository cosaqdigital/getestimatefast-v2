# GFSEO-007 — Playbook de Produção, Atualização e Governança de Conteúdo

Versão: 1.0
Status: Approved
Sistema: GF-OS
Projeto: GetEstimateFast

## Objetivo
Padronizar todo o ciclo de vida do conteúdo, desde o planejamento até a atualização contínua.

## Escopo
Páginas comerciais, locais, pilares, blog, FAQs, guias, comparativos, estudos de caso e conteúdos institucionais.

## Fluxo
Pesquisa → Intenção → Briefing → Produção → Revisão SEO → Revisão Técnica → Revisão Editorial → Publicação → Monitoramento → Atualização.

## Briefing Obrigatório
Objetivo, keyword principal, keywords secundárias, intenção, cluster, persona, CTA, links internos e prioridade.

## Estrutura
Title, Meta Description, URL, H1, conteúdo estruturado, FAQ, CTA, links internos, Schema e Breadcrumb.

## Uso de IA
Permitida para apoio à produção, nunca substituindo revisão humana e aprovação.

## Checklist
Conteúdo original, SEO validado, revisão técnica, links internos, Schema, CTA e aprovação GF-OS.

## Auditoria
Revisão periódica para identificar conteúdos desatualizados, duplicados, canibalização e oportunidades.

## Indicadores
Impressões, cliques, CTR, sessões, conversão, leads, atualizações e crescimento por cluster.

## Conclusão
Este playbook estabelece o padrão oficial para criação, manutenção e evolução do conteúdo do GetEstimateFast.
