# GFSEO-002 — Arquitetura Estratégica de Keywords

Versão: 1.0
Status: Approved
Sistema: GF-OS
Projeto: GetEstimateFast

## Objetivo
Definir a arquitetura oficial de palavras-chave do GetEstimateFast para orientar pesquisa, classificação, priorização e expansão do SEO.

## Princípios
- Uma intenção por página
- Uma keyword principal
- Crescimento por clusters
- Foco na intenção do usuário
- Evitar canibalização

## Taxonomia
Comercial, Transacional, Informacional, Navegacional, Comparativa, Emergencial e Local.

## Estrutura
Serviço → Subserviço → Cidade → Intenção.

## Mapeamento
Cada keyword deve possuir intenção, cluster, serviço, cidade, página destino, prioridade e status.

## Critérios
Demanda, relevância, potencial de leads, alinhamento estratégico e possibilidade de conteúdo útil.

## Long-tail
Priorizar palavras-chave específicas para acelerar autoridade e conversão.

## Estratégia Geográfica
Cada combinação Serviço + Cidade constitui uma oportunidade independente.

## Canibalização
Antes de criar nova página, validar se já existe conteúdo equivalente.

## Indicadores
Cobertura de clusters, cobertura geográfica, Top 3, Top 10, CTR e conversão.

## Conclusão
A arquitetura de keywords estabelece a base operacional para o crescimento orgânico sustentável do GetEstimateFast.
