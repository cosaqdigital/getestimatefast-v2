# GFSEO-008 — Playbook de Auditorias Técnicas de SEO

Versão: 1.0
Status: Approved
Sistema: GF-OS
Projeto: GetEstimateFast

## Objetivo
Estabelecer o processo oficial para auditorias técnicas de SEO, garantindo rastreabilidade, indexação, performance e qualidade contínua.

## Escopo
Website, páginas comerciais, páginas locais, blog, hubs, Quote Flows e infraestrutura.

## Frequência
Auditoria rápida semanal, completa mensal, estrutural trimestral e extraordinária após mudanças relevantes.

## Processo
Coletar dados → Identificar problemas → Classificar severidade → Priorizar → Corrigir → Validar → Documentar.

## Áreas
Indexação, rastreamento, performance, HTML, Schema.org, segurança e links internos.

## Checklist
Sitemap, robots, canonicals, Core Web Vitals, HTTPS, headings, metas, Schema e links.

## Ferramentas
Google Search Console, GA4, Lighthouse, PageSpeed Insights, Screaming Frog e validadores de Schema.

## KPIs
Páginas indexadas, erros de cobertura, Core Web Vitals, erros 404, redirecionamentos, páginas órfãs e evolução das correções.

## Conclusão
Este playbook estabelece o padrão oficial para manter a saúde técnica e a evolução sustentável do SEO do GetEstimateFast.
