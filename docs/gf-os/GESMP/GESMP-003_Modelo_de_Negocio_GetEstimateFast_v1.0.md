# GESMP-003 — Modelo de Negócio do GetEstimateFast

**Versão:** 1.0
**Status:** Approved
**Sistema:** GF-OS
**Projeto:** GetEstimateFast

## 1. Objetivo
Definir o modelo de negócio do GetEstimateFast e como a plataforma cria valor para usuários, parceiros e para o projeto.

## 2. Resumo Executivo
O GetEstimateFast é uma plataforma de geração de oportunidades para serviços residenciais. Seu papel é conectar proprietários de imóveis a profissionais locais por meio de uma experiência simples e eficiente.

## 3. Problema Resolvido
Para proprietários: dificuldade em encontrar profissionais confiáveis.
Para parceiros: dificuldade em gerar novos clientes de forma previsível.

## 4. Proposta de Valor
### Proprietários
- Solicitação rápida de estimativas.
- Facilidade para encontrar profissionais locais.

### Parceiros
- Recebimento de oportunidades qualificadas.
- Aumento da previsibilidade comercial.

## 5. Modelo Operacional
Usuário → Google → Página de Serviço → Quote Flow → Lead → Parceiro.

## 6. Fontes de Receita
- Venda de leads qualificados.
- Assinaturas (futuro).
- Destaques para parceiros.
- Publicidade.
- Afiliados.
- Ferramentas premium.

## 7. Público-Alvo
Proprietários de imóveis, investidores, administradores e profissionais locais.

## 8. Diferenciais Competitivos
Foco local, experiência simples, conteúdo útil, SEO sustentável e melhoria contínua.

## 9. Crescimento
1. Consolidar Riverview.
2. Expandir cidades vizinhas.
3. Escalar para novas regiões.

## 10. Indicadores
Leads, CTR, sessões orgânicas, conversão, parceiros ativos e desempenho por serviço.

## 11. Riscos
Dependência da busca orgânica, qualidade dos parceiros, mudanças de algoritmo e baixa conversão.

## 12. Princípios Comerciais
Transparência, foco no usuário, relacionamento sustentável e comunicação responsável.

## Conclusão
O modelo de negócio foi concebido para crescer de forma sustentável, preservando a experiência do usuário e a geração de oportunidades qualificadas.
